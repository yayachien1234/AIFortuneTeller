# AI 算命系統

這是一個簡易的 AI 算命系統，讓使用者輸入個人資料，然後顯示當日運勢和建議。系統提供了友善的介面，並透過隨機生成的分析與建議，讓使用者得到適時的方向和啟發。

## 功能

1. **資料輸入**：
   - 使用者需填入姓名、性別、出身年月日、今日日期，並選擇貓派或狗派。
   - 完成輸入後按下按鈕進入下一步。

2. **神諭時刻**：
   - 顯示當日運勢和建議，內容從預設的建議中隨機選取。
   - 按下按鈕可返回資料輸入頁面，並清除之前輸入的資料。

## 錯誤處理

- 如果欄位留空、性別或貓狗選擇未正確填寫，會在對應欄位顯示錯誤訊息。
- 日期欄位的輸入會被視為有效，不進行額外的格式檢查。
